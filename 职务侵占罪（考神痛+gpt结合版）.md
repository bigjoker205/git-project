# 职务侵占罪法考主观题高分讲义

## 一、规范定义与本质重构

**法条定位**：《中华人民共和国刑法》第二百七十一条

**规范定义**：公司、企业或者其他单位的工作人员，利用职务上的便利，将本单位财物非法占为己有，数额较大的，处三年以下有期徒刑或者拘役，并处罚金；数额巨大的，处三年以上十年以下有期徒刑，并处罚金；数额特别巨大的，处十年以上有期徒刑或者无期徒刑，并处罚金。

**本质重构**：职务侵占罪的本质在于公司、企业或其他单位的工作人员，利用其职务所赋予的主管、管理、经手本单位财物的便利条件，将本单位财物非法占为己有，侵犯单位财产所有权的行为。其关键在于行为人利用职务便利实施非法占有行为，且达到数额较大标准。

## 二、构成要件精细拆解

### 1. 客体

侵犯的法益是公司、企业或其他单位的财产所有权。

### 2. 客观方面

- **行为方式**：利用职务上的便利，实施侵吞、窃取、骗取或者以其他手段非法占有本单位财物的行为。
    
    - **侵吞**：将自己主管、管理、经手的单位财物直接据为己有，如财务人员将单位账户资金转入自己银行卡。
    - **窃取**：利用职务上的便利，以违反单位意思的方法，将自己合法管理、使用的本单位财物非法占为己有，如仓库保管员监守自盗。
    - **骗取**：利用职务上的便利，使用欺骗的方法，非法占有本单位财物，如公司管理人员虚报公司财务支出，将多报销的费用占为己有。
    
- **危害结果**：达到数额较大标准，根据相关司法解释，职务侵占“数额较大”的起点是6万元。

### 3. 主体

特殊主体，即公司、企业或者其他单位的工作人员。包括：

- 公司股东、法定代表人、总经理、部门经理等管理人员；
- 财务、会计、出纳、仓库管理员等直接经手单位财物的人员；
- 临时工、实习生等只要实际负责管理、经手单位财物的人员。

需注意，国有公司、企业或者其他国有单位中从事公务的人员和国有公司、企业或者其他国有单位委派到非国有公司、企业以及其他单位从事公务的人员，利用职务上的便利侵占公共财物的，应当认定为贪污罪。

### 4. 主观方面

必须出于故意，且具有非法占有的目的，即行为人明知是单位的财物，却主动、故意地将其占为己有，并且有“永久占有”的意图。

## 三、罪名区分与争议焦点

### 1. 职务侵占罪 vs 贪污罪

表格

|区分标准|职务侵占罪|贪污罪|
|---|---|---|
|主体|公司、企业或者其他单位的工作人员|国家工作人员|
|犯罪对象|本单位的财物，既可能是公共财物，也可能是私有财物|只能是公共财物|
|数额要求|“数额较大”“数额巨大”的数额起点，按照贪污罪相对应的数额标准规定的二倍、五倍执行|贪污或者受贿数额在三万元以上不满二十万元的，应当认定为“数额较大”；贪污或者受贿数额在二十万元以上不满三百万元的，应当认定为“数额巨大”|

### 2. 职务侵占罪 vs 挪用资金罪

表格

| 区分标准 | 职务侵占罪             | 挪用资金罪             |
| ---- | ----------------- | ----------------- |
| 犯罪客体 | 侵犯的是公司、企业等单位财物所有权 | 侵犯的是单位资金的使用权      |
| 犯罪对象 | 本单位财物，包括资金及财物     | 仅指本单位资金           |
| 犯罪目的 | 以非法占有为目的，即永久占有    | 以非法使用为目的，准备日后归还   |
| 客观行为 | 将单位财物占为己有         | 挪用单位资金归个人使用或借贷给他人 |
|      |                   |                   |

### 3. 职务侵占罪 vs 盗窃罪

表格

|区分标准|职务侵占罪|盗窃罪|
|---|---|---|
|主体|特殊主体，即公司、企业或者其他单位的工作人员|一般主体|
|行为方式|利用职务上的便利，将自己主管、管理、经手的单位财物非法占为己有|秘密窃取他人财物，与职务无关|
|犯罪对象|本单位的财物|可以是任何公私财物|

## 四、典型案例三段论分析

### 【案例】

杨某在担任某医药销售企业分店店长期间，利用职务便利，以“团购”的名义向公司申请名贵药材，转而以低于标价的价格售卖给回收二手商品的商家，货款未入公司账户，而是直接流入其个人支付宝与银行卡用于还债。为应付盘点，她在系统里上演“空盘入库”戏码，将已私自卖掉的商品重新录入库存系统，制造“账实相符”的假象。当公司催促她追回“团购”货款时，她用“拆东墙补西墙”的招数，从公司申请更多货物，售卖后，将部分钱款作为“回款”交给区域经理，企图蒙混过关。经查，杨某的行为导致4家药企累计遭受118万余元的损失。

### 【三段论分析】

- **大前提（构成要件）** ：职务侵占罪要求行为人主观上有非法占有本单位财物的故意，客观上利用职务上的便利，将本单位财物非法占为己有，数额较大。
- **小前提（案件事实）** ：杨某作为医药销售企业分店店长，利用其负责管理店铺药品的职务便利，以“团购”名义申请药品并私自售卖，将货款据为己有，且通过各种手段掩盖其侵占行为，累计侵占单位财物118万余元，数额巨大。
- **结论（法律评价）** ：杨某的行为符合职务侵占罪的构成要件，构成职务侵占罪。

## 五、转化形态与特殊问题

### 1. 转化形态

- **向贪污罪转化**：国有公司、企业或者其他国有单位中从事公务的人员和国有公司、企业或者其他国有单位委派到非国有公司、企业以及其他单位从事公务的人员，利用职务便利侵占本单位财物，应按贪污罪定罪处罚。
- **向其他犯罪转化**：在实施职务侵占过程中使用暴力手段造成他人人身伤害等严重后果，可能同时符合故意伤害罪等其他犯罪构成，司法实践中可能根据想象竞合或牵连犯等原则，从一重罪处罚。

### 2. 特殊问题

- **个体工商户经营者**：个体工商户不属于职务侵占罪中的“单位”，经营者处置自己的财产，不构成此罪。
- **挂靠、外包人员**：没有和单位建立劳动关系，只是挂靠在单位名下，或做外包服务，哪怕接触单位财物，也不算“内部人”，不构成职务侵占罪。
- **挂名人员**：挂名股东、挂名高管，没有实际参与公司经营，也没有管理、经手单位财物的权限，不算符合主体，不构成职务侵占罪。

## 六、加重情节系统审查

### 1. 法定加重情节

- 数额巨大（100万元以上），处三年以上十年以下有期徒刑，并处罚金；
- 数额特别巨大，处十年以上有期徒刑或者无期徒刑，并处罚金。

### 2. 酌定加重情形

- 动机特别卑劣；
- 事先预谋充分；
- 犯罪后无悔改表现；
- 侵占单位重要财物，严重影响单位正常经营。

### 3. 常见考查点

- 如何认定“数额巨大”“数额特别巨大”？
- 加重情节与量刑的关系。

## 七、常见失分陷阱

### 1. 陷阱：将“挪用资金罪”误判为“职务侵占罪”

- **原因**：未准确把握行为人的主观故意内容，挪用资金罪以非法使用为目的，准备日后归还；职务侵占罪以非法占有为目的，不打算归还。
- **正确做法**：必须审查行为人的主观故意是非法占有还是临时挪用。

### 2. 陷阱：忽视“利用职务上的便利”要素

- **原因**：仅关注行为形式，未分析是否利用职务上的便利。
- **正确做法**：必须审查行为是否利用了职务上的便利，即行为人是否基于工作职责能够占用、处分本单位财物。

### 3. 陷阱：混淆“职务便利”与“工作便利”

- **原因**：将因工作关系熟悉环境、容易接近财物、便于利用他人疏忽等便利条件，误认为是职务便利。
- **正确做法**：必须区分“职务便利”与“工作便利”，职务便利是指利用自己主管、管理、经手本单位财物的职权或职责便利，而工作便利仅指因工作关系熟悉环境等便利条件。

### 4. 陷阱：遗漏“数额较大”的标准

- **原因**：未准确掌握职务侵占罪的数额要求。
- **正确做法**：必须审查侵占的数额是否达到“数额较大”的标准，即6万元以上。

## 八、记忆压缩模块

### 【记忆口诀】

职务侵占单位人，利用职务占财物。

## 九、作答采分点清单

1. **法条引用（3分）** ：准确引用刑法第二百七十一条。
2. **构成要件分析（5分）** ：完整分析四要件，包括客体、客观方面、主体、主观方面。
3. **主观故意认定（4分）** ：明确行为人具有非法占有的目的。
4. **客观行为描述（3分）** ：准确描述利用职务便利实施的侵占行为方式。
5. **罪名区分（5分）** ：与易混淆罪名（如贪污罪、挪用资金罪、盗窃罪）的对比分析。
6. **法律评价（3分）** ：基于事实和法律的法律适用结论。
7. **三段论推理（2分）** ：大前提、小前提、结论完整。

**总分：25分**

## 十、核心抓手总结

职务侵占罪的核心抓手是：特殊主体（单位工作人员）+ 利用职务便利 + 非法占有本单位财物 + 数额较大，答题时必须围绕“职务便利”和“非法占有”两个核心展开，准确区分与其他相似罪名的界限。
