这个选题极具战略价值——敲诈勒索罪作为财产犯罪的核心罪名，不仅是法考主观题的高频考点，其与抢劫罪、诈骗罪的边界辨析更是实务中的难点。我将从法考应试逻辑出发，为你构建完整的知识体系。

<summary>一、敲诈勒索罪的构成要件（法考主观题答题框架）</summary>

### 1. 客体要件：复杂客体

- **主要客体**：公私财物的所有权
- **次要客体**：他人的人身权利或其他权益（如名誉、隐私）

### 2. 客观要件：威胁或要挟的行为结构

plaintext

```plaintext
实施威胁/要挟行为 → 被害人产生恐惧心理 → 基于恐惧心理处分财产 → 行为人取得财产 → 被害人遭受财产损失
```

#### 关键考点辨析：

- **威胁内容的多样性**：可以是暴力威胁（如伤害、杀害），也可以是毁坏财物、揭发隐私、破坏名誉等非暴力威胁
- **威胁的实现方式**：可以是明示，也可以是暗示；可以直接向被害人发出，也可以通过第三人转达
- **威胁的时间性**：不要求当场实现，也可以是将来实现

### 3. 主体要件：一般主体

- 刑事责任年龄：16周岁以上（注意：抢劫罪是14周岁）

### 4. 主观要件：直接故意，且具有非法占有目的

- 必须具有非法占有他人财物的目的，排除行使正当权利的行为（如索要合法债务）

<summary>二、法考高频考点：此罪与彼罪的辨析</summary>

### 1. 敲诈勒索罪与抢劫罪的区别

表格

|**区别点**|**敲诈勒索罪**|**抢劫罪**|
|---|---|---|
|**暴力程度**|一般不具有当场实施的紧迫性，被害人有一定的选择余地|暴力程度高，具有当场实施的紧迫性，被害人丧失选择余地|
|**取财时间**|可以当场取财，也可以事后取财|必须当场取财|
|**威胁内容**|可以是当场实现，也可以是将来实现|必须是当场实现|

### 2. 敲诈勒索罪与诈骗罪的区别

- **被害人处分财产的原因不同**：
    
    - 敲诈勒索罪：基于恐惧心理处分财产
    - 诈骗罪：基于错误认识处分财产
    
- **竞合处理**：如果行为同时具有欺骗和胁迫性质，被害人既产生错误认识又产生恐惧心理，则从一重罪处罚
    

### 3. 敲诈勒索罪与[[绑架罪]]的区别

- **行为对象不同**：
    
    - 敲诈勒索罪：直接对被害人实施威胁或要挟
    - 绑架罪：控制人质后，向第三人提出不法要求

<summary>三、法考主观题答题技巧：案例分析的得分要点</summary>

### 1. 答题模板：

plaintext

```plaintext
被告人的行为构成敲诈勒索罪。理由如下：
1. 主体要件：被告人达到刑事责任年龄，具有刑事责任能力。
2. 主观要件：被告人具有非法占有他人财物的直接故意。
3. 客观要件：被告人实施了威胁/要挟行为，使被害人产生恐惧心理，并基于恐惧心理处分了财产，被告人取得了财产，被害人遭受了财产损失。
4. 客体要件：被告人的行为侵犯了被害人的财产所有权和人身权利。

综上所述，被告人的行为符合敲诈勒索罪的构成要件，应当以敲诈勒索罪定罪处罚。
```

### 2. 得分要点：

- **准确认定行为结构**：必须完整阐述威胁→恐惧→处分→取财的因果链条
- **区分此罪与彼罪**：如果案例中存在与其他罪名的混淆点，必须进行辨析
- **量刑情节考量**：注意是否存在既遂、未遂、数额巨大、数额特别巨大等量刑情节

</details>

<summary>四、法考客观题高频陷阱</summary>

### 1. 正当权利行使与敲诈勒索的界限

- **关键判断标准**：是否具有非法占有目的
- **合法行为**：索要合法债务、举报违法行为并要求奖励等
- **非法行为**：以揭发隐私为要挟索要超出合法债务的财物

### 2. 敲诈勒索罪的既遂与未遂

- **既遂标准**：行为人取得财物
- **未遂标准**：被害人未处分财产或行为人未取得财物

### 3. 敲诈勒索罪的数额标准

- **数额较大**：2000元以上（各地可能有不同标准）
- **数额巨大**：3万元以上
- **数额特别巨大**：30万元以上

<summary>五、经典案例分析（法考真题改编）</summary>

**案例：** 甲得知乙与丙有不正当男女关系，遂以向乙的妻子揭发此事为由，要求乙支付10万元“封口费”。乙因害怕妻子得知此事，被迫向甲支付了10万元。

**问题：** 甲的行为构成何罪？

**答案要点：**

1. 甲的行为构成敲诈勒索罪。
2. 甲以揭发乙的隐私为要挟，使乙产生恐惧心理，并基于恐惧心理处分了财产，符合敲诈勒索罪的构成要件。
3. 甲的行为不构成诈骗罪，因为乙不是基于错误认识处分财产，而是基于恐惧心理处分财产。
4. 甲的行为不构成抢劫罪，因为甲没有当场使用暴力或暴力威胁，乙有一定的选择余地。
