# 帮助犯罪分子逃避处罚罪：法考难点深度解析

## 一、罪名核心与立法逻辑

帮助犯罪分子逃避处罚罪（《刑法》第417条）是渎职罪中的难点罪名，其立法目的在于规制国家机关工作人员利用职务便利实施的"内鬼"行为，与普通公民实施的窝藏、包庇行为形成鲜明区分。

### 1. 构成要件拆解

表格

| 构成要件   | 核心要点                                      | 法考命题陷阱                     |
| ------ | ----------------------------------------- | -------------------------- |
| **主体** | 特殊主体：负有查禁犯罪活动职责的国家机关工作人员，包括辅警等实际履行查禁职责的人员 | 非查禁职责的国家机关工作人员（如后勤人员）不构成本罪 |
| **主观** | 直接故意，明知对方是犯罪分子，希望帮助其逃避处罚                  | 过失不构成本罪，可能构成玩忽职守罪          |
| **客观** | 利用职务便利，实施通风报信、提供便利等行为                     | 未利用职务便利的帮助行为可能构成窝藏、包庇罪     |
| **客体** | 司法机关查禁犯罪的正常活动 + 国家机关工作人员职务行为的廉洁性          | 与徇私枉法罪、窝藏包庇罪的竞合关系          |

### 2. 关键概念辨析

- **"犯罪分子"的扩大解释**：包括实施了犯罪行为但尚未被发现的人、已被立案侦查的犯罪嫌疑人、被采取强制措施的被告人和正在服刑的罪犯。即使后续被判无罪，通风报信行为仍可能构成本罪。
- **"通风报信"的表现形式**：口头、书面、电子信息等方式，将查禁犯罪活动的时间、地点、人员、方案等信息告知犯罪分子。
- **"提供便利"的多样性**：提供隐藏处所、财物，帮助逃匿；协助伪造、毁灭证据；利用职务便利拖延或阻碍案件调查等。

## 二、法考高频考点与易混淆罪名区分

### 1. 与徇私枉法罪的区分

表格

|区分维度|帮助犯罪分子逃避处罚罪|徇私枉法罪|
|---|---|---|
|**主体**|有查禁犯罪职责的国家机关工作人员|司法工作人员（侦查、检察、审判、监管职责）|
|**行为方式**|通风报信、提供便利|明知是无罪的人而使他受追诉、明知是有罪的人而故意包庇不使他受追诉，或者在刑事审判活动中故意违背事实和法律作枉法裁判|
|**竞合处理**|想象竞合，择一重罪论处|优先适用徇私枉法罪|

**法考真题示例**：(2010年卷二第6题)公安人员甲向犯罪分子乙通风报信助其逃避处罚。甲、乙成立共同犯罪？

**答案**：不成立共同犯罪。甲成立帮助犯罪分子逃避处罚罪，乙以其实施的犯罪单独定罪处罚。

### 2. 与窝藏、包庇罪的区分

表格

|区分维度|帮助犯罪分子逃避处罚罪|窝藏、包庇罪|
|---|---|---|
|**主体**|特殊主体（有查禁职责的国家机关工作人员）|一般主体（任何具有刑事责任能力的自然人）|
|**行为方式**|利用职务便利实施通风报信、提供便利|提供隐藏处所、财物，帮助逃匿；作虚假证明包庇|
|**客体**|国家机关查禁犯罪的正常履职活动|国家正常司法活动|
|**竞合处理**|想象竞合，择一重罪论处|优先适用帮助犯罪分子逃避处罚罪|

**法考真题示例**：(2017年卷二第63题)公安局协警闫某，在协助抓捕行动中，向领导黑社会性质组织的李某通风报信，导致李某等主要犯罪分子潜逃。闫某构成帮助犯罪分子逃避处罚罪？

**答案**：正确。协警属于实际履行查禁职责的人员，符合本罪主体要求。

### 3. 与徇私舞弊不移交刑事案件罪的区分

表格

|区分维度|帮助犯罪分子逃避处罚罪|徇私舞弊不移交刑事案件罪|
|---|---|---|
|**主体**|有查禁犯罪职责的国家机关工作人员|行政执法人员|
|**行为方式**|通风报信、提供便利|对应当移交司法机关追究刑事责任的不移交，情节严重|
|**竞合处理**|想象竞合，择一重罪论处|优先适用徇私舞弊不移交刑事案件罪|

## 三、罪数形态与法考命题模型

### 1. 想象竞合犯

- **与帮助毁灭、伪造证据罪**：帮助、示意犯罪分子隐匿、毁灭、伪造证据，逃避处罚，构成本罪，同时构成帮助毁灭、伪造证据罪，想象竞合。
- **与妨害作证罪**：帮助、示意犯罪分子串供、翻供，构成本罪，同时构成妨害作证罪，想象竞合。
- **与掩饰、隐瞒犯罪所得罪**：公安人员对盗窃、抢劫的机动车辆，非法提供机动车辆牌证，或者为其取得机动车辆牌证提供便利，构成本罪，同时构成掩饰、隐瞒犯罪所得罪，想象竞合。

### 2. 数罪并罚

- **受贿罪与帮助犯罪分子逃避处罚罪**：国家机关工作人员实施渎职犯罪并收受贿赂，同时构成受贿罪的，除刑法另有规定外，以渎职犯罪和受贿罪数罪并罚。（《关于办理渎职刑事案件适用法律若干问题的解释（一）》第三条）

**典型案例**：常州市公安局经开区分局街道派出所三级警长张陶俊收受盗窃案犯罪嫌疑人贿送的现金人民币1.8万元，试图包庇两人逃避刑事处罚，同时构成受贿罪和帮助犯罪分子逃避处罚罪，数罪并罚。

### 3. 共犯认定

- 与犯罪分子事先通谋，事后帮助逃避处罚的，以共同犯罪论处。
- 单纯的通风报信、提供便利行为，不构成共同犯罪，单独构成帮助犯罪分子逃避处罚罪。

## 四、法考应试技巧与得分要点

### 1. 客观题解题思路

- **主体判断优先**：首先判断行为人是否属于"有查禁犯罪活动职责的国家机关工作人员"。
- **职务关联判断**：判断帮助行为是否利用了职务便利。
- **罪名竞合处理**：若同时构成多个罪名，需根据想象竞合、法条竞合等原则选择正确答案。

### 2. 主观题答题框架

- **定罪三段论**：主体→客观行为→主观故意→构成要件符合性。
- **罪数分析**：是否存在想象竞合、数罪并罚等情形。
- **量刑情节**：自首、立功、认罪认罚等从宽情节，以及受贿后帮助逃避处罚等从重情节。

### 3. 易错点警示

- **"犯罪分子"的扩大解释**：即使后续被判无罪，通风报信行为仍可能构成本罪。
- **辅警的主体资格**：实际履行查禁职责的辅警可构成本罪。
- **职务便利的认定**：未利用职务便利的帮助行为可能构成窝藏、包庇罪。

## 五、历年法考真题回顾与解析

### 1. 2010年卷二第6题

**题目**：公安人员甲向犯罪分子乙通风报信助其逃避处罚。甲、乙成立共同犯罪？

**答案**：不成立共同犯罪。甲成立帮助犯罪分子逃避处罚罪，乙以其实施的犯罪单独定罪处罚。

### 2. 2017年卷二第63题

**题目**：公安局协警闫某，在协助抓捕行动中，向领导黑社会性质组织的李某通风报信，导致李某等主要犯罪分子潜逃。闫某构成帮助犯罪分子逃避处罚罪？

**答案**：正确。协警属于实际履行查禁职责的人员，符合本罪主体要求。

### 3. 2023年试题

**题目**：环保部门的执法人员在执法检查时发现甲实施污染环境罪，故意不移交刑事案件侦查机关，构成本罪？

**答案**：构成徇私舞弊不移交刑事案件罪，而非帮助犯罪分子逃避处罚罪。

## 六、备考策略与总结

### 1. 重点掌握内容

- 本罪的构成要件，特别是主体范围和行为方式。
- 与徇私枉法罪、窝藏包庇罪、徇私舞弊不移交刑事案件罪的区分。
- 想象竞合犯和数罪并罚的适用情形。

### 2. 记忆口诀

- **主体**：查禁职责国家机关，辅警履职也可算。
- **行为**：通风报信提供便利，利用职务是关键。
- **区分**：徇私枉法是司法人员，窝藏包庇是一般主体。
