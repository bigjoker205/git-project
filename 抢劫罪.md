## 一、抢劫罪的法条基础

**《刑法》第263条**规定：
> 以暴力、胁迫或者其他方法抢劫公私财物的，处三年以上十年以下有期徒刑，并处罚金；有下列情形之一的，处十年以上有期徒刑、无期徒刑或者死刑，并处罚金或者没收财产……

**法条结构**：基本犯（3-10年）+ 8种加重情节（10年以上至死刑）

## 二、构成要件拆解（应试核心）

### （一）客体要件：复杂客体
- **主要客体**：财产所有权
- **次要客体**：人身权利（生命、健康权）

> **应试要点**：抢劫罪是侵犯财产罪中唯一侵犯双重客体的犯罪，这是区分于盗窃、诈骗的关键。

### （二）客观要件：手段行为 + 目的行为

| 要素       | 内容         | 应试陷阱             |
| -------- | ---------- | ---------------- |
| **手段行为** | 暴力、胁迫或其他方法 | 必须达到"[[压制反抗]]"程度 |
| **目的行为** | 当场取财       | 强调时空关联性          |

#### 1. "暴力"的认定标准
- **本质**：最广义的有形力，包括殴打、捆绑、伤害、杀害
- **程度要求**：足以**压制被害人反抗**（不是轻微暴力）
- **应试要点**：若暴力程度明显轻微，可能构成**抢夺罪**或**敲诈勒索罪**

#### 2. "胁迫"的认定
- **即时性**：以当场实施暴力相威胁
- **当场性**：威胁必须能够当场实现
- **对比记忆**：敲诈勒索罪的威胁不具有当场实现性

#### 3. "其他方法"的认定
- **典型情形**：灌醉、麻醉、催眠
- **核心特征**：使被害人**不知反抗或不能反抗**
- **法考高频考点**：利用被害人**自身状态**（如熟睡、醉酒）取财，是否属于"其他方法"？
  - **答案**：不是！必须是**行为人主动制造**的不能反抗状态

> **经典真题模型**：甲见乙醉酒，临时起意取走乙钱包 → **盗窃罪**；甲故意灌醉乙后取财 → **抢劫罪**

### （三）主体要件
- **一般主体**：年满14周岁即可（注意：盗窃需16周岁）
- **特殊考点**：14-16周岁的人可构成抢劫罪，但不能构成转化型抢劫（后述）

### （四）主观要件
- **直接故意** + **非法占有目的**
- **目的实现时间**：当场取得财物（包括当场取得财产性利益）

## 三、加重情节：8种情形（必背！）

| 情节 | 关键词 | 应试要点 |
|------|--------|----------|
| 1. 入户抢劫 | "户"的认定 | 供家庭生活、相对隔离；**不包括**机关、企事业单位 |
| 2. 在公共交通工具上抢劫 | 运营中的公共交通工具 | 小型出租车不算；拦截后抢劫算 |
| 3. 抢劫银行或其他金融机构 | 经营资金、有价证券、客户资金 | 抢劫办公用品不算 |
| 4. 多次抢劫或数额巨大 | 3次以上/参照盗窃标准 | 多次指不同时间、不同地点 |
| 5. 抢劫致人重伤、死亡 | **结果加重犯** | 包括故意或过失；必须是抢劫行为本身导致 |
| 6. 冒充军警人员抢劫 | 身份冒充 | 真军警人员显示身份抢劫，从重处罚但不适用本项 |
| 7. 持枪抢劫 | "枪"的认定 | 真枪+展示；仿真枪不算 |
| 8. 抢劫军用物资或抢险救灾物资 | 特定物资 | 必须是"物资"本身 |

> **背诵口诀**：入户公交抢银行，多次巨额人伤亡，冒充军警持枪抢，军用救灾物资抢

## 四、转化型抢劫（第269条）：法考重中之重

**法条**：《刑法》第269条
> 犯盗窃、诈骗、抢夺罪，为窝藏赃物、抗拒抓捕或者毁灭罪证而**当场使用暴力或者以暴力相威胁**的，依照抢劫罪定罪处罚。

### 构成要件拆解

| 要素 | 具体内容 | 应试陷阱 |
|------|----------|----------|
| **前提** | 犯盗窃、诈骗、抢夺罪（行为，不要求既遂） | 实施行为即可，不要求数额较大 |
| **目的** | 三种特定目的：窝藏赃物、抗拒抓捕、毁灭罪证 | 必须是这三种目的之一 |
| **行为** | 当场使用暴力或以暴力相威胁 | "当场"包括现场及追捕过程中 |
| **暴力程度** | 足以压制反抗 | 若轻微暴力，可能不构成转化 |

### 三大核心争议点（每年必考）

#### 1. "当场"的认定
- **空间**：实施盗窃等行为的现场
- **时间**：被追捕的连续过程中
- **判断标准**：**社会一般观念**认为仍处于紧迫状态

#### 2. 14-16周岁的人能否构成转化型抢劫？
- **司法解释立场**：**不能**
- **理由**：转化型抢劫以盗窃等为前提，14-16周岁的人对盗窃等不负刑责，不能转化为抢劫
- **例外**：若暴力行为本身构成故意伤害罪（重伤或死亡），则单独定罪

#### 3. 携带凶器抢夺的认定（第267条第2款）
- **法律拟制**：携带凶器抢夺 = 抢劫罪
- **"凶器"认定**：枪支、爆炸物、管制刀具 + **为犯罪而携带的其他器械**
- **关键限制**：**不能显示、不能使用**，若显示或使用，直接认定为抢劫罪（非转化）

## 五、抢劫罪与相关罪名的界限（对比表）

| 对比罪名 | 核心区别 | 应试要点 |
|----------|----------|----------|
| **vs 抢夺罪** | 是否对人实施暴力 | 抢夺是对物暴力，抢劫是对人暴力 |
| **vs 敲诈勒索罪** | 暴力的当场性、威胁的实现时间 | 抢劫：暴力当场实现；敲诈：暴力将来实现 |
| **vs 绑架罪** | 取财对象、时间 | 抢劫：向被害人当场取财；绑架：向第三人取财，不限时间 |
| **vs 故意伤害罪** | 主观目的、行为结构 | 抢劫是取财目的+暴力手段；伤害是伤害目的 |
| **vs 寻衅滋事罪** | 主观目的、暴力程度 | 抢劫有明确取财目的；寻衅滋事是逞强耍横 |

### 特殊情形：抢劫罪与绑架罪的竞合
- **核心区分**：向谁要钱？何时要钱？
  - 甲控制乙，逼乙当场交出财物 → **抢劫罪**
  - 甲控制乙，向乙的家人要钱，何时交付不限 → **绑架罪**
- **法考陷阱**：若控制被害人后，向其本人和家属同时索财，**从一重罪**（通常定绑架罪）

## 六、罪数问题：一罪与数罪

### （一）法条竞合与想象竞合
1. **抢劫罪 vs 故意杀人罪**
   - 抢劫过程中**为取财而杀人** → **抢劫罪一罪**（结果加重犯）
   - 抢劫**既遂后**灭口杀人 → **抢劫罪 + 故意杀人罪**，数罪并罚
   - 杀人后**临时起意**取财 → **故意杀人罪 + 盗窃罪**

2. **抢劫罪与强奸罪**
   - 先抢劫，后强奸 → **数罪并罚**
   - 先强奸，后抢劫 → **数罪并罚**（无吸收关系）

### （二）加重情节的罪数
- **抢劫致人重伤、死亡**：结果加重犯，**一罪**
- **抢劫后灭口**：另起犯意，**数罪并罚**

## 七、法考应试技巧总结

### （一）做题三步法
1. **判断行为结构**：是否存在"压制反抗+当场取财"？
2. **检查加重情节**：8种情形是否具备？（特别是"入户""公共交通工具""致人重伤死亡"）
3. **排查转化情形**：是否属于269条转化型抢劫？或267条携带凶器抢夺？

### （二）高频陷阱归纳
| 陷阱情形 | 正确结论 | 错误选项 |
|----------|----------|----------|
| 利用被害人自身醉酒状态取财 | 盗窃罪 | 抢劫罪 |
| 14-16周岁转化型抢劫 | 不定抢劫罪（暴力另定罪） | 定抢劫罪 |
| 抢劫后灭口 | 抢劫+故意杀人，数罪并罚 | 抢劫一罪 |
| 真军警显示身份抢劫 | 从重处罚，不适用"冒充"情节 | 适用"冒充军警"加重情节 |
| 持仿真枪抢劫 | 不定"持枪抢劫" | 定"持枪抢劫" |

### （三）主观题答题模板
若遇到抢劫罪案例分析，建议按以下结构作答：

```
1. 构成要件分析：
   - 客观：甲实施了XX行为，属于暴力/胁迫/其他方法，足以压制被害人反抗，并当场取得财物
   - 主观：甲具有非法占有目的

2. 是否构成加重情节：
   - 若涉及入户/公交/金融机构/致人重伤死亡等，需具体分析是否符合法定标准

3. 是否涉及转化：
   - 若存在盗窃/诈骗/抢夺前提，需分析是否符合269条转化条件

4. 罪数处理：
   - 是否存在数罪并罚或择一重罪情形
```

## 八、典型案例演练

**案例**：甲潜入乙家盗窃，被乙发现。甲为抗拒抓捕，将乙打成轻伤后逃离。甲刚出门即被巡逻警察丙发现，甲又对丙实施暴力，将丙打成重伤后逃跑。

**问题**：甲的行为如何定性？

**解析**：
1. **第一阶段**：盗窃转化为抢劫（269条），针对乙的暴力构成转化型抢劫
2. **第二阶段**：对丙的暴力属于"当场"延续（被追捕过程中），但此时针对的是抓捕者而非原盗窃对象
3. **结论**：整体认定为**抢劫罪一罪**，致人重伤属于加重情节（"抢劫致人重伤"），处十年以上有期徒刑。
